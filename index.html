3D диагностика стопы
Изготовление ортопедических стелек

Проект Insole занимается проведением диагностики стопы методом 3D сканирования отпечатка стопы и изготовлением индивидуальных ортопедических стелек с методом фрезеровки на станках с числовым программным управлением (ЧПУ).

Поддержка при ...

... продольном плоскостопии.

... поперечном плоскостопии
(hallux valgus).

... комбинированном плоскостопии.

... усталости и болях в своде стопы.

Коррекция при ...

... вальгусной деформации.

... варусной деформации.

... полой стопе.

... приведении стопы.

... последствиях травм.

... последствиях хирургического вмешательства.

Разгрузка при ...

... диабетической стопе.

... плантарном фасциите
(пяточной шпоре).

... болезни Шинца.
